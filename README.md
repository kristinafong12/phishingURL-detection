# Phishing URL Detector

A tool that detects phishing URLs using structural features and a simple machine learning model.

## Features
- Detects suspicious URLs based on patterns
- Uses Logistic Regression for classification
- Simple and fast to run

## Example
Input: http://paypal-login-secure.com  
Output: ❌ WARNING Phishing Site
